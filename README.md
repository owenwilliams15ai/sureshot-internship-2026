## What this is

This repo is the portfolio for my summer 2026 internship at Sureshot, a B2B MarTech company that builds software and integrations for Oracle Eloqua. The program is an 8-module curriculum focused on building, deploying, and operating AI agents using Anthropic's Claude stack.

The goal isn't traditional data analytics — it's to come out the other end as an **agent operator**: someone who can design, build, and run AI agents that do real work for a real business.

---

## Stack

- **Claude Code** — agent-enabled IDE / CLI
- **Claude Agent SDK** — agent runtime
- **Model Context Protocol (MCP)** — how agents talk to external systems
- **Anthropic API** — programmatic builds (Python)
- **Python** — pandas, numpy, requests, and whatever else the project needs
- **Git / GitHub** — version control for everything

## 8-Module Curriculum

| Module | Focus |
|--------|-------|
| 1 | AI Fluency foundations — Chat vs. Cowork vs. Code decision framework |
| 2 | Prompt engineering and structured output — building a reusable prompt library |
| 3 | MCP fundamentals — how agents connect to external systems |
| 4 | Competitor research agent — daily automated intelligence pipeline to Slack |
| 5 | Applied automation — Lori-defined internal Sureshot project |
| 6 | Custom MCP server — exposing a Sureshot data source as a programmable interface |
| 7 | Production polish — error handling, retries, logging, evals, safety documentation |
| 8 | Capstone — portfolio wrap-up, recorded demo, final documentation |
